# AI 投資研究助理

一套「建議制」的個人投資研究系統:每個交易日自動研究你的觀察清單,產出買/賣/持有建議與完整理由,寫成結構化日誌,並更新可視化儀表板。**系統不會下單** — 所有決定由你確認、由你執行。

## 每天自動做的事

1. **研究(Research)**:抓取觀察清單的行情(Yahoo Finance,備援 Stooq),計算 SMA/RSI/MACD/布林通道/量能等技術指標;抓取每檔的新聞頭條並做情緒初評。
2. **建議(Advise)**:綜合技術訊號 + 新聞情緒 + 你的持倉風控(停損線、部位上限、現金保留),對每檔給出「建議買進/加碼/續抱/減碼/賣出/暫不動作」與逐條理由。
3. **日誌(Journal)**:把每一個決策(包含「今天決定不動作」)寫進 `journal/YYYY-MM-DD.md` 與機器可讀的 `journal/journal.jsonl`。
4. **發掘(Discover)**:掃描市場漲幅榜/成交活躍榜,列出值得研究的潛力候選(不等於建議買進)。
5. **教學(Teach)**:依當天實際觸發的訊號,挑一堂對應的投資課(內建 20+ 堂繁中課程)。
6. **儀表板(Dashboard)**:更新 `docs/index.html` — 單一檔案、支援手機與深色模式,可直接發佈成 GitHub Pages 網站。

## 專案結構

```
config/
  watchlist.json    ← 觀察清單(直接編輯)
  portfolio.json    ← 你的持倉與現金(直接編輯;目前是示範持倉)
  settings.json     ← 風控參數與建議門檻
advisor/            ← 核心模組(資料/指標/新聞/策略/發掘/日誌/教學/儀表板)
run.py              ← 主流程,一行指令跑完整輪
journal/            ← 每日決策日誌(Markdown + JSONL)
docs/index.html     ← 可視化儀表板(GitHub Pages 直接吃這個資料夾)
output/             ← data.json 與每日快照(資產走勢由此累積)
.github/workflows/daily.yml ← 每個交易日自動執行的排程
```

## 快速開始

本機執行(需 Python 3.10+):

```bash
pip install -r requirements.txt
python run.py            # 抓真實資料
python run.py --demo     # 網路不可用時,以「示範資料」展示功能(會明確標示)
```

雲端自動化部署 → 看 **SETUP.md**(手把手,約 10 分鐘)。

## 日常使用

- **看結果**:開儀表板網頁(GitHub Pages 網址),或直接讀 `journal/` 下的日誌。
- **改觀察清單**:編輯 `config/watchlist.json` 的 `tickers` 陣列。
- **更新持倉**:你在券商實際成交後,把股數/成本更新到 `config/portfolio.json`(`_示範` 欄位請一併刪除)。也可以直接把成交結果告訴 Claude,請它幫你改。
- **調風控**:`config/settings.json` — 停損 %、單一部位上限、買進/賣出門檻等都在這裡。

## 建議的閱讀順序(每天 3 分鐘)

1. 儀表板「總覽」:今天有沒有動作建議?
2. 有建議 → 點進「訊號」看理由與風險提示,自行判斷要不要執行。
3. 掃一眼「新聞」與「潛力雷達」。
4. 讀「每日一課」,累積知識。

## 免責聲明

本系統為個人研究工具,所有輸出由公開資料與規則化訊號自動產生,僅供參考,**不構成投資建議**。市場有風險,過去的訊號表現不代表未來;交易決策與盈虧請自行負責,重大財務決定請諮詢持牌專業人士。
